# Payboard Dashboard

Responsive financial dashboard built from the approved desktop and mobile designs.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`. On mobile, use the bottom navigation to switch between overview, analytics, and wallet views.

## Stack

Next.js App Router, TypeScript, Tailwind CSS, shadcn-style components, Context API, TanStack React Query, Recharts, and Lucide icons.
